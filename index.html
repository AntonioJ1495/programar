Acerca de Python

Python es un lenguaje de programación interpretado, de alto nivel y multiparadigma creado por Guido van Rossum en 1991.

Características principales:

  • Tipado dinámico
  • Multiplataforma
  • Orientado a objetos
  • Gran colección de librerías
  • Código abierto

Áreas de aplicación:

Web Ciencia de Datos IA Automatización Juegos
