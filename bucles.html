Control de bucles

Python ofrece palabras clave para controlar el flujo de los bucles:

break Termina el bucle inmediatamente
continue Salta a la siguiente iteraci√≥n
else Se ejecuta si el bucle termina normalmente (sin break)

Ejemplo: break y continue

# Ejemplo de break
for numero in range(10):
    if numero == 5:
        break
    print(numero)

# Ejemplo de continue
for numero in range(10):
    if numero % 2 == 0:
        continue
    print(numero)
